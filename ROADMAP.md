# PulseWatch Roadmap (v0.1)

Wedge: **SaaS + agencies** — *Competitor Pricing / Packaging Radar*.
Alert MVP channel: **Discord**.

## Workflow (enforced)
- GitHub-first / PR-only + Draft PR early
- Preflight for tasks >30 min
- MUNCH→Decide→Commit (max 2 munch cycles)
- Skill-or-Justify (≤90s)
- Evidence/Assumptions gate; STOP guessing
- Output Contract (Deliverable/Evidence/Skill used/Next Action/Owner)

## v0.1 Milestone (high-level)
1) PR1: Repo scaffolding + DB schema + artifact storage layout
2) PR2: Capture pipeline (Playwright) + normalization
3) PR3: Diff + scoring + taxonomy + evidence pack
4) PR4: Alerting (Discord) + weekly digest
5) PR5: Minimal UI (alerts feed + evidence view)
6) PR6: Noise control UX (ignore selectors + dismiss-as-noise learning)
7) PR7: CI/CD + Sentry + Playwright smoke gates
8) PR8: Pilot onboarding (seed competitor set) + metrics dashboard (dismiss rate, time-to-detect)

## Success metrics for v0.1 pilot
- <10% alerts dismissed as noise
- Median time-to-detect < 12h
- Evidence pack present on 100% alerts
